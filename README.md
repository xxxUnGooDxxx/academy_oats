# Академия ОАТС

Обучающий курс-тренажёр по сервисам **Облачной АТС** (билайн) для менеджеров с нуля. Объясняет простыми словами, что такое телефония и ОАТС, зачем сервисы бизнесу и как выявлять потребность у клиента. После каждого блока — мини-тест.

## Методика

«Воронка понимания» от боли бизнеса к решению. Каждый блок построен по схеме:
**Зачем → Что это (+ схема звонка) → Кейс по отрасли → Как продавать (SPIN-скрипт) → Тест.**

## Программа (8 блоков)

0. Что такое телефония
1. Что такое облачная АТС и зачем бизнесу
2. Приём и распределение звонков
3. Тарифы и из чего складывается цена
4. Запись, контроль и аналитика
5. Интеграции и CRM
6. Маркетинг и привлечение
7. Автоматизация обзвона

## Стек

- **Vite + React** (SPA), маршрутизация — `react-router-dom` (HashRouter, чтобы работать на GitHub Pages).
- **mermaid** — схемы прохождения звонка.
- **lucide-react** — иконки. Шрифт Inter, фирменный стиль билайн.
- Прогресс и результаты тестов — в `localStorage`.

## Разработка

```bash
npm install
npm run dev      # локальный сервер
npm run build    # сборка в dist/
npm run preview  # предпросмотр сборки
```

## Структура

```
src/
  data/         контент: блоки, цены, экзамен, мета курса
    blocks/     по файлу на блок + index.js
  components/   Layout, Quiz, Mermaid, Icon
  pages/        Home, Block
  lib/          прогресс (localStorage)
public/assets/  логотип билайн, иконки
.github/workflows/deploy.yml  авто-деплой на GitHub Pages
```

## Деплой

Пуш в `main` → GitHub Action собирает проект и публикует на GitHub Pages.
В настройках репозитория: **Settings → Pages → Source → GitHub Actions**.
`base: './'` в `vite.config.js` обеспечивает работу на любом подпути Pages.

## Результаты в Google Таблицы

Сайт после сдачи экзамена отправляет результат в Google Таблицу (через Apps Script, без сервера). Подключение: вставить URL веб-приложения в `src/config.js` или в `.env` (`VITE_SHEETS_ENDPOINT`). Пошаговая инструкция и код — в `docs/google-apps-script.md`. Пока URL не задан — результаты хранятся только в localStorage.

## Шпаргалки PDF

На экране «Шпаргалка» в каждом блоке есть кнопка **«Скачать PDF»** — открывает печать браузера с фирменным одностраничным макетом (главное + цены + вопросы клиенту). Печатный макет — `src/components/CheatsheetDoc.jsx`, показывается только при печати (`@media print`).

## Источник цен

Прайс-лист «Облачная АТС» (2026, 01-26). Цены информационные, с НДС (кроме Голосового робота и Колл-центра PRO), без надбавки 1,2%.
